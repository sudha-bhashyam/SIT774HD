OTP Login Feature ‚Äì Developer Tutorial

This section explains how the secure OTP (One-Time Password) login system was implemented using Node.js, Express, crypto, and Nodemailer.

üí° How it Works

  • User enters their email address in the login form
  • The server generates a 6-digit OTP using crypto.randomInt() for security
  • The OTP is stored with a timestamp in memory and sent to the user via email using Nodemailer
  • User enters the OTP on the verification page
  • The server checks if the OTP matches and is still valid (2-minute expiry window)
  • If valid, a session is created using express-session

üîß Technologies Used

  • crypto ‚Äì for secure OTP generation
  • nodemailer ‚Äì to send OTPs to user emails
  • express-session ‚Äì to maintain user login state
  • dotenv ‚Äì for storing email credentials and secrets
